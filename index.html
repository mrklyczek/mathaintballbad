Welcome To My New Site!

This is the first time that I've fully put together an interactive website without using a creator like Weebly, Wix, or Jigsy. Using just html, css, and java, I'm very proud of what I've accomplished. But, I need to give credit where credit is due!


  • w3schools
    This place is amazing! Not only can I find pretty much anything related to web development, but I can progress through tutorials and really LEARN something. Oh, and it's FREE!

  • CodePen
    While I love the service that CodePen offers- a way to code in different langauges in the same workspace, what really makes it so awesome is the people. Ranging from professionals to hobbiests, coders/developers SHARE with the world. It has an awesome sense of community built into it.

  • Trinket Code
    I'm actually building my website using Trinket's editor. It is simple enough for a beginner like me to use, but powerful enough for even the best coders.

  • Honorable Mentions
    • Google Fonts
    • CodeMyUI
    • CSS Newbie
    • CSS Tricks
